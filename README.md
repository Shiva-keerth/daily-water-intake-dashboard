# Daily Water Intake Analytics Dashboard

Interactive Streamlit dashboard to analyze daily water intake patterns using demographic and lifestyle factors.

## Objective
Analyze how water intake varies by age, gender, weight, physical activity level, weather conditions, and hydration level.

## Tech Stack
- Python
- Pandas, NumPy
- Matplotlib
- Streamlit

## Key Features
- Dataset overview (preview, dtypes, missing values)
- Individual analysis (age, gender, weight, intake, activity, weather, hydration)
- GroupBy analysis (average intake by categories and multi-variable comparisons)
- KPI insights (total records, average intake, low intake count)

## Dataset
Dataset sourced from Kaggle (not included in this repository).

## Outcome
Built an interactive dashboard to explore hydration trends and identify low-intake groups.
