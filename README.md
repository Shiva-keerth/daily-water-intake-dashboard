# 💧 Daily Water Intake Analytics Dashboard

An interactive Streamlit dashboard to analyze daily water intake patterns based on demographic and lifestyle factors using Python and Pandas.

---

## 🎯 Objective

Analyze how water intake varies by age, gender, weight, physical activity level, weather conditions, and hydration level to identify low-consumption groups and generate actionable health insights.

---

## 🚀 Key Features

- 📋 Dataset overview — preview, data types, missing value check
- 👤 Individual analysis — age, gender, weight, activity, weather & hydration
- 📊 GroupBy analysis — average intake by categories & multi-variable comparisons
- 📈 KPI insights — total records, average intake, low intake count
- 💡 Identifies low-consumption groups based on demographic patterns

---

## 🛠️ Tech Stack

| Tool | Purpose |
|---|---|
| Python | Core programming language |
| Pandas & NumPy | Data manipulation & GroupBy analysis |
| Matplotlib | Data visualization |
| Streamlit | Interactive dashboard UI |
| Kaggle | Dataset source |

## ▶️ How to Run

1. Clone the repository
```bash
git clone https://github.com/Shiva-keerth/daily-water-intake-dashboard.git
cd daily-water-intake-dashboard
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Download the dataset from Kaggle and place it in the project folder

4. Run the Streamlit app
```bash
streamlit run water_intake_dashboard.py
```
